## Skill Selection Process

### Step 1: Select Product and Role
```
Product Type: [Oracle Fusion ▼]
Primary Role: [Senior Developer ▼]
```
→ Skills section appears

### Step 2: Add a Skill
```
Skill:        [Select a skill...▼]
              ↓ (click dropdown)
              
Available Skills:
├─ JavaScript        (if not already selected)
├─ TypeScript        (if not already selected)
├─ Python            (if not already selected)
├─ React             (if not already selected)
└─ ... (23 more technical + 10 soft skills)
```

### Step 3: Select Rating
```
Rating: ☆ ☆ ☆ ☆ ☆
        1 2 3 4 5
        
Default: ★★★☆☆ (3 stars)
```

### Step 4: Add Skill to List
```
[Add Skill] button → Skill added to selected list
```

### Step 5: View Selected Skills
```
Selected Skills (5)

┌─────────────────────────────────┐
│ React                [Remove]   │
│ Proficiency: ★★★★☆            │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ TypeScript          [Remove]    │
│ Proficiency: ★★★★★             │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ Leadership          [Remove]    │
│ Proficiency: ★★★☆☆             │
└─────────────────────────────────┘
```

### Step 6: Modify Ratings
Click stars in skill card to update rating:
```
┌─ Selected Skills ──────────────────────┐
│                                        │
│ React              [Remove]            │
│ Proficiency: ★★★☆☆ ← Click to change │
│              1 2 3 4 5                 │
│                                        │
└────────────────────────────────────────┘
```

### Step 7: Save Profile
```
[Create Profile] or [Update Profile] button
→ Skills saved with profile data
```

## Behavior Notes

1. **Conditional Display**: Skills section only appears after both Product AND Role are selected
2. **Duplicate Prevention**: System prevents adding the same skill twice
3. **Interactive Ratings**: Click stars to adjust 1-5 proficiency level
4. **Real-time Filtering**: Skill dropdown only shows skills not yet added
5. **Easy Management**: Remove button quickly deletes skills from selection
6. **Contextual Help**: Helper text shows which product/role skills are for
7. **Visual Feedback**: Grid layout with skill cards makes selections clear

## CSS Styling

The SkillSelector component uses:
- `skill-selector-container`: Main wrapper
- `skill-add-section`: Input area with dropdown, rating, and add button
- `rating-stars`: Interactive star rating (1-5)
- `rating-stars-preview`: Preview stars while selecting
- `selected-skills-list`: Container for selected skills
- `skills-grid`: Grid layout of skill cards
- `skill-card`: Individual skill item display
- `btn-add-skill`: Add skill button styling
- `btn-remove-skill`: Remove skill button styling

Styles are defined in: `react-frontend/src/styles/SkillSelector.css`
