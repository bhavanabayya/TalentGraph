# Social Media Links Feature - Quick Reference

## What's New?

Candidates can now add professional and social media links (LinkedIn, GitHub, Portfolio, etc.) to their profile. Links are displayed as clickable hyperlinks on the profile overview.

## Where to Find It

### For Editing
- **Edit Profile Page**: Scroll down to see "Social Media & Professional Links" section
- **Dashboard**: Edit from the profile management section

### For Viewing
- **Profile Overview**: Links appear as clickable buttons with platform icons
- **Public Profile**: Links visible to other users viewing your profile

## How to Use

### Adding a Link

1. Go to "Edit Profile" page
2. Find "Social Media & Professional Links" section
3. Select platform from dropdown
4. Paste your full URL
   - Example: `https://linkedin.com/in/yourname`
   - Example: `https://github.com/yourname`
5. (Optional) Add custom display name
6. Click "+ Add Link" or press Enter
7. Link appears in your list
8. Click "Save Profile" button

## Troubleshooting

**URL not accepted?**
- Ensure it starts with `https://` or `http://`
- Check for correct spelling
- Try copying directly from browser address bar

**Link not saving?**
- Click "Save Profile" button at bottom of form
- Check for error messages
- Refresh page to verify

**Want to see saved links?**
- Go back to Edit Profile
- Links appear in your list
- Profile Overview shows clickable links

## FAQs

**Can I have multiple links?**  
Yes! Add as many as you want.

**Can I reorder my links?**  
Currently no, but you can remove and re-add in desired order.

**Do links open in same tab?**  
No, they open in a new tab so you don't lose your profile.

**Can recruiter see my links?**  
Yes! Links are visible on your public profile.

**What if I don't want to add links?**  
Completely optional! Just skip this section.

## Privacy Notes

- Links are public and visible to everyone viewing your profile
- Only add links you want professionally shared
- Profile is HTTPS secure
- No tracking on your personal website/portfolio links
