## â“ FAQ

**Q: Can I have multiple profiles?**
A: Yes! Create as many as you want for different products/roles.

**Q: Can I edit a profile after creating it?**
A: Yes! Just click "Edit" on the profile card.

**Q: What if I delete a profile by accident?**
A: Unfortunately it's permanent. You'll need to recreate it.

**Q: Can I change the product after creating a profile?**
A: Not directly. You'd need to delete and recreate it.

**Q: What's the difference between "save" and "update"?**
A: "Save" = Creating new profile. "Update" = Editing existing profile.

**Q: Will these profiles be seen by companies?**
A: Yes! Companies can view your profiles when browsing candidates.
